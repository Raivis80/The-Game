Game rules

  1. Click on squares as fast you can.
  2. One clicked Square adds one point.
  3. The game starts with three lives.
  4. Each Missed target will substract one life and black object will spawn instead if clicked, ends the game altogether The black object will reset every 100p streak
  5. A streak of 50p adds one life.
  6. As you progress in the game, speed and target count will increase.
  7. Score as many points as you can.

Have fun playing the game.
